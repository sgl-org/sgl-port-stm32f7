# STM32F7xx SGL PORT TEMPLATE

This project demonstrates the operation of sgl on the STM32F7xx platform.

## How to compile

```
# apt install gcc-arm-none-eabi
# ./build.sh
```

## Run benchmark

Flash the generated image `build/examples/demo/demo.hex` to stm32 chip.

```
Render: 226fps
```
